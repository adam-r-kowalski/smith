# Key Technologies

[Tree-Sitter](https://tree-sitter.github.io/tree-sitter/) can parse the Smith language and give me syntax highlighting across all IDEs.

[LLVM](https://llvm.org/) allows me to generate code for various architectures including X86, X86-64, PowerPC, PowerPC-64, ARM, Thumb, SPARC, Alpha, CellSPU, MIPS, MSP430, SystemZ, WebAssembly, and XCore.

[wgpu](https://wgpu.rs/) lets me generate graphics using Vulkan, Metal, DirectX 12, OpenGL ES, WebGPU, and WebGL2.

[OpenXLA](https://github.com/openxla) supports generation for AWS Tranium & Inferentia, AMD GPU, Arm CPUs, Cerebras WSE, Google TPU, Graphcore IPU, NVIDIA GPU, and x86 CPU.

[Language Server Protocol (LSP)](https://microsoft.github.io/language-server-protocol/) provides features in every IDE such as go to definition, go to references, autocomplete, and documentation.

[Debug Adapter Protocol (DAP)](https://microsoft.github.io/debug-adapter-protocol/) offers debugging and running your program and tests in every IDE.
