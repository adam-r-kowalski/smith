Introducing Smith

  • Compiled language for high performance on CPUs and GPUs
  • Advanced static type system tracking element types and dimensions
  • Shape checking at compile time
  • Source-to-source automatic differentiation at compile time
  • Immutability by default for safe parallelism without locks
  • Generalized Einstein summation notation for linear algebra

Open Source Strategy

  • Smith is open-source, licensed under the Mozilla Public License 2.0 (MPL 2.0)
  • Facilitates commercial use, allowing seamless integration into products
  • Encourages sharing improvements with the community
  • Enriches the ecosystem and fosters greater innovation

Roadmap and Milestones
